## 使用方法

### AstrBot

1. 下载 `Light-skill.zip`
2. 在 AstrBot 中导入：Skills 广场 → 自定义 Skills → 新建 → 上传 zip
3. 等待审核通过后即可使用

### 其他平台

将 `PERSONA.md` 的内容复制到对应平台的角色设定中即可。
